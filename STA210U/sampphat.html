Sampling Distribution of \(\hat{p}\)

Estimating the Population Proportion

The sample proportion \(\hat{p}\) estimates the population proportion \(p\):

  • Population Proportion: \( p = \frac{\text{\# in population with characteristic}}{\text{\# in population}} \)
  • Sample Proportion: \( \hat{p} = \frac{\text{\# in sample with characteristic}}{n} \)

Key Properties

  • \(\mu_{\hat{p}} = p\)
    The average value of \(\hat{p}\) across all possible samples equals \(p\).
  • \(\sigma_{\hat{p}} = \sqrt{\frac{p(1 - p)}{n}}\)
    The standard error of \(\hat{p}\) is \(\sqrt{\frac{p(1 - p)}{n}}\).

Normal Approximation

For large \(n\): \[ Z = \frac{\hat{p} - p}{\sqrt{\frac{p(1 - p)}{n}}} \] is approximately standard normal.

Examples

Example 1: Stats Class Pass Rates

A university’s true pass rate for its introductory statistics course is 80%. A random sample of 80 students is taken.

  1. Find \(\mu_{\hat{p}}\).
  2. Find \(\sigma_{\hat{p}}\).
  3. Using the normal approximation, find the probability that \(\hat{p}\) is between 0.75 and 0.85.

1. \(\mu_{\hat{p}} = p = 0.80\)

2. \(\sigma_{\hat{p}} = \sqrt{\frac{p(1 - p)}{n}} = \sqrt{\frac{0.80 \cdot (1 - 0.80)}{80}} = \sqrt{\frac{0.80 \cdot 0.20}{80}} = \sqrt{\frac{0.16}{80}} = \sqrt{0.002} \approx 0.0447\)

3. Using \( Z = \frac{\hat{p} - p}{\sigma_{\hat{p}}} \):
For \(\hat{p} = 0.75\): \( Z = \frac{0.75 - 0.80}{0.0447} \approx -1.12 \)
For \(\hat{p} = 0.85\): \( Z = \frac{0.85 - 0.80}{0.0447} \approx 1.12 \)
P(-1.12 < Z < 1.12) ≈ 0.737 (from standard normal tables, P(Z < 1.12) ≈ 0.8685, P(Z < -1.12) ≈ 0.1314, so 0.8685 - 0.1314 = 0.737).

Example 2: Equine Vaccination Rates

The true proportion of horses vaccinated against equine influenza is 60%. In a survey of 200 horse owners, 160 report their horses are vaccinated.

  1. Calculate \(\hat{p}\).
  2. Calculate \(\sigma_{\hat{p}}\) using the true \(p\).
  3. Using the normal approximation, find the probability that \(\hat{p}\) is greater than 0.65.

1. \(\hat{p} = \frac{160}{200} = 0.8\)

2. \(\sigma_{\hat{p}} = \sqrt{\frac{p(1 - p)}{n}} = \sqrt{\frac{0.60 \cdot (1 - 0.60)}{200}} = \sqrt{\frac{0.60 \cdot 0.40}{200}} = \sqrt{\frac{0.24}{200}} = \sqrt{0.0012} \approx 0.0346\)

3. Using \( Z = \frac{\hat{p} - p}{\sigma_{\hat{p}}} \):
For \(\hat{p} = 0.65\): \( Z = \frac{0.65 - 0.60}{0.0346} \approx 1.45 \)
P(Z > 1.45) ≈ 1 - P(Z < 1.45) ≈ 1 - 0.9265 ≈ 0.0735 (from standard normal tables).
