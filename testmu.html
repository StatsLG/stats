Using the Standard Normal Distribution

In hypothesis testing for the population mean (\(\mu\)), we use the test statistic:

\[ Z = \frac{\bar{x} - \mu}{\sigma / \sqrt{n}} \]

Here, \(\bar{x}\) is the sample mean, \(\mu\) is the hypothesized population mean, \(\sigma\) is the population standard deviation, and \(n\) is the sample size. This test statistic, \(Z\), follows a standard normal distribution when conducting a test for \(\mu\), assuming the population is normally distributed or the sample size is large (\(n \geq 30\)).

The \(Z\)-statistic allows us to calculate the p-value, which is the probability of observing a test statistic as extreme as, or more extreme than, the one calculated, under the null hypothesis (\(H_0\)). We use techniques from standard normal distribution probability calculations (introduced in section III.C. and revisited in section IV.A. for sampling distributions) to find the p-value.

To perform a hypothesis test for \(\mu\), follow these steps:

  1. State the null (\(H_0\)) and alternative (\(H_a\)) hypotheses.
  2. Calculate the test statistic (\(Z\)).
  3. Determine the p-value using the standard normal distribution.
  4. Compare the p-value to the level of significance (\(\alpha\)):
    • If \(p\)-value \(\leq \alpha\), reject \(H_0\).
    • If \(p\)-value \(> \alpha\), fail to reject \(H_0\).
  5. Interpret the result in the context of the problem.

The p-value depends on the type of test:

  • Left-tailed test (\(H_a: \mu < \mu_0\)): p-value = \(P(Z < z)\).
  • Right-tailed test (\(H_a: \mu > \mu_0\)): p-value = \(P(Z > z) = 1 - P(Z < z)\).
  • Two-tailed test (\(H_a: \mu \neq \mu_0\)): p-value = \(2 \times P(Z > |z|)\).

Example 1

A consumer protection agency wants to prove that packages of Post Grape Nuts have a mean weight of less than 24 oz. Use the following information to test this claim:

\(\alpha = 0.05\), \(n = 100\), \(\bar{x} = 23.94\), \(\sigma = 0.13\)

  1. State the hypotheses.

    Null Hypothesis: \(H_0: \mu \geq 24\)

    Alternative Hypothesis: \(H_a: \mu < 24\) (left-tailed test)

  2. Calculate the test statistic.

    \[ Z = \frac{\bar{x} - \mu}{\sigma / \sqrt{n}} = \frac{23.94 - 24}{0.13 / \sqrt{100}} = \frac{-0.06}{0.13 / 10} = \frac{-0.06}{0.013} \approx -4.615 \]

  3. Determine the p-value.

    For a left-tailed test, p-value = \(P(Z < -4.615)\).

    Using standard normal tables, \(P(Z < -4.615) \approx 0.000002\) (since \(-4.615\) is far in the left tail).

  4. Make a decision and interpret the result.

    Since p-value \(\approx 0.000002 < 0.05 = \alpha\), reject \(H_0\).

    Interpretation: There is sufficient evidence at the 5% significance level to conclude that the mean weight of Post Grape Nuts packages is less than 24 oz.

Example 2

It is desired to show the mean weight of a metal component is greater than 4.5 oz. Use the following information to conduct a hypothesis test:

\(\alpha = 0.05\), \(n = 10\), \(\bar{x} = 4.59\), \(\sigma = 0.504\)

  1. State the hypotheses.

    Null Hypothesis: \(H_0: \mu \leq 4.5\)

    Alternative Hypothesis: \(H_a: \mu > 4.5\) (right-tailed test)

  2. Calculate the test statistic.

    \[ Z = \frac{\bar{x} - \mu}{\sigma / \sqrt{n}} = \frac{4.59 - 4.5}{0.504 / \sqrt{10}} = \frac{0.09}{0.504 / 3.162} = \frac{0.09}{0.1594} \approx 0.565 \]

  3. Determine the p-value.

    For a right-tailed test, p-value = \(P(Z > 0.565) = 1 - P(Z < 0.565)\).

    Using standard normal tables, \(P(Z < 0.565) \approx 0.714\), so p-value = \(1 - 0.714 = 0.286\).

  4. Make a decision and interpret the result.

    Since p-value \(\approx 0.286 > 0.05 = \alpha\), fail to reject \(H_0\).

    Interpretation: There is insufficient evidence at the 5% significance level to conclude that the mean weight of the metal component is greater than 4.5 oz.
