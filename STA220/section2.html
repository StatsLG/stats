Section 2 — Methods for Describing Sets of Data

Descriptive statistics via graphs: bar charts for categorical data and several displays for quantitative data (stem‑and‑leaf, frequency tables, histograms).

Graphing Qualitative Data

Use a bar chart for qualitative (categorical) variables. Bars compare frequencies or percentages for categories; order along the x‑axis is arbitrary.

  • Bars are uniform width and evenly spaced; x‑axis has no numeric meaning.
  • Height/length of bar encodes the value; viewers notice tallest bars first.

Graphing Quantitative Data

Goal: reveal the distribution and identify extreme values (outliers) and overall shape.

Extreme value (outlier): an observation separated from the rest by a noticeable gap.

Stem-and-Leaf

  • Split values into stems (leading digits) and leaves (trailing digit).
  • Lets you recover original data while seeing shape and possible outliers.

Frequency Distributions

  • Group quantitative data into classes (5–15 classes, equal width, non‑overlapping).
  • Useful for large data sets; original values are not retained after grouping.
  • Class midpoint: halfway between class boundaries.

Histograms

  • x‑axis is quantitative; bar width has numeric meaning and bars touch.
  • Displays the frequency distribution; x‑axis ticks often show class midpoints.
