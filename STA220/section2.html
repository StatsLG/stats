Section 2 — Methods for Describing Sets of Data

Bar Graph — Important Characteristics

  • Use only for qualitative (categorical) data. The x-axis is not numeric, so the order of bars doesn’t carry numeric meaning.
  • Bar length encodes the quantity being compared (frequency or percent).
  • Viewers tend to notice the tallest bars.
  • Bars should be uniform width and uniformly spaced (x-axis is non-numeric).

Graphing Quantitative Data

Goal: examine the distribution of a single quantitative variable—identify extreme values and the overall shape.

Extreme Values (Outliers)

Extreme value (outlier): an observation separated from the rest of the data by a noticeable margin.

Example

Data: 14, 15, 17, 21, 60, 23, 25 → the extreme value is 60.

Impact: a single extreme income (a billionaire) can inflate the sample mean for “average income.”

Stem-and-Leaf

  • Splits values into “stems” (leading digits) and “leaves” (trailing digit).
  • Organizes and groups data but still lets us recover original values.
  • Good for spotting extreme values and assessing shape.

Here, 201 is an extreme value (separated by at least one empty class).

Frequency Distributions

  • Arrange quantitative data into class groupings (typically 5–15 classes).
  • Classes should be equal width, and non-overlapping.
  • Useful for large data sets; original individual values are not retained.
  • Class midpoint = halfway between class boundaries.

Histograms

  • x-axis is quantitative; bar width has numeric meaning and bars touch.
  • A histogram is a graphical picture of the frequency distribution.
  • x-axis labels typically show class midpoints; y-axis shows frequency.

Compare with the stem-and-leaf: they tell a similar story about extremes and shape (rotating the stem-and-leaf can resemble a histogram).

Shapes of Distributions

  • Symmetrical: mirror halves about the center.
  • Uniform: every class has equal frequency.
  • Skewed: one tail extends farther. Direction of skewness is the side of the longer tail.
    • Skewed right: tail stretched to the right.
    • Skewed left: tail stretched to the left.
  • Bimodal: two distinct peaks.

Summation Notation (Σ)

For values \(x_1, x_2, \dots, x_n\), the sum is written \(\sum_{i=1}^{n} x_i\). Here, \(x\) denotes the data, \(n\) the sample size, and \(i\) indexes observations.

Example — Siblings

Data: 2, 3, 1, 3

  • \(\sum x = 2+3+1+3 = 9\)

Useful Properties (\(c\) a constant)

  • \(\sum (c\,x) = c \sum x\)
  • \(\sum c = n\,c\)
  • \(\sum (x + c) = \sum x + \sum c = \sum x + n\,c\)

Numerical Measures of Central Tendency

Sample Mean

\(\bar{x} = \dfrac{\sum x}{n}\)

Examples

  • Data 2,3,1,3 → \(\bar{x} = 9/4 = 2.25\) (reasonable here; two values below and two above).
  • Add 10 → Data 2,3,1,3,10 → \(\bar{x} = 19/5 = 3.8\). Now the mean is pulled by the extreme 10, so it’s less representative.

Mean characteristics: sensitive to extremes; not necessarily an attainable data value.

Sample Median

Order data; if \(n\) is odd, the middle value; if \(n\) is even, the average of the two middle values.

  • 2,3,1,3 → ordered 1,2,3,3 → \(\tilde{x} = (2+3)/2 = 2.5\)
  • 2,3,1,3,10 → ordered 1,2,3,3,10 → \(\tilde{x} = 3\)

Median characteristics: not sensitive to extremes; half of data below and half above the median.

Mean vs Median and Skewness

  • If \(\bar{x} > \tilde{x}\): right-skewed.
  • If \(\bar{x} < \tilde{x}\): left-skewed.
  • If \(\bar{x} = \tilde{x}\): symmetric.

Sample Mode

Most frequent value(s). 2,3,1,3 → mode = 3 (unchanged if data become 2,3,1,3,10). Mode can be unstable (may not exist or may be multiple), but it can be used with qualitative data (e.g., “average student is female” means the mode is female).

Sample Midrange

\(\dfrac{\text{Low}+\text{High}}{2}\). For 2,3,1,3 → midrange \(=\,(1+3)/2 = 2\). For 2,3,1,3,10 → midrange \(=\,(1+10)/2 = 5.5\). Midrange depends entirely on extremes and is rarely a good measure (except under perfect symmetry).
