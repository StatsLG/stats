Confidence Interval for \(\mu\) Using the Standard Normal Distribution

Remember that \(\mu_{\bar{x}} = \mu\) and \(\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}}\).

Therefore, when estimating \(\mu\):

\(E = z \left(\frac{\sigma}{\sqrt{n}}\right)\)

\(\bar{x} \pm E\)

A sample of 100 visa accounts was taken to estimate the population mean unpaid balance of all visa accounts.

Confidence Interval for \(\mu\) Using the t-Distribution

As we study inferential statistics, the main goal is understanding the process which is consistent even when the mathematics gets more complicated. When doing confidence intervals by hand we will always use the standard normal distribution, but you may have noticed a problem with using \(E = z \left(\frac{\sigma}{\sqrt{n}}\right)\). The issue is when we are estimating the population mean, \(\mu\), we will not know the population standard deviation, \(\sigma\) (the only way to know this quantity would be to have the population, and if we had the population we could just calculate the mean). In statistics, when we do not know a value, we estimate it. So, in this case we can estimate the population standard deviation, \(\sigma\), with the sample standard deviation, \(s\). The ramification of replacing \(\sigma\) by \(s\) is the sampling distribution is no longer standard normal. The sampling distribution will instead be a t-distribution.

The t-distribution is still continuous and symmetric about 0, but it is more variable and a slightly different shape than the standard normal distribution. Basically, we are estimating another quantity so there is more error in the estimate. The confidence interval formula for \(\mu\) using the t-distribution follows:

\(E = t \left(\frac{s}{\sqrt{n}}\right)\)

\(\bar{x} \pm E\)

As you can see the only change in the formula is \(\sigma\) being replaced by \(s\) which means the \(z\) becomes a \(t\). As stated above you will not need to do t-distribution problems by hand. However, when doing the lab assignment you will compute confidence intervals using the t-distribution. Of course, in these problems the software will do the math.
