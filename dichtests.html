Dichotomous Tests

A dichotomous test is a test with only two possible results. Typically, these results are refered to as "positive" and "negative." A positive result means the test determined whatever it was testing for was true or present. Common examples include roadside sobriety tests, home pregnancy tests, and cancer screenings.

The importance of these tests extends to hypothesis testing in inferential statistics.

False Positive Rate (FPR)

The probability of a test concluding positive given the actual condition is negative.

Specificity

The probability of a test concluding negative given the actual condition is negative.

False Negative Rate (FNR)

The probability of a test concluding negative given the actual condition is positive.

Sensitivity

The probability of a test concluding positive given the actual condition is positive.
