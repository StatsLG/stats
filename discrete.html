Valid Discrete Probability Distributions

Intro to Probability Distributions

First, it is important to make sure that we distinguish between a discrete and continuous random variable. While the idea of a probability distribution is the same in both cases, the way we have to look at the probability distribution is different. We will focus on the idea of probability for a discrete random variable here.

Discrete Random Variable – potential values are separated points on the number line; there is a finite or countable number of values (outcomes are typically counted)

Continuous Random Variable – potential values fall in an interval on the number line; there is an infinite number of values (outcomes are typically measured)

A discrete random variable will include a finite or countable number of outcomes. Therefore, we assign probabilities to each value of the random variable.

Probability distribution – the collection of \( P(x) \) values for all possible x values

For a discrete probability distribution to be valid, it must satisfy the following:

  1. The variable, x, must be discrete
  2. \( 0 \leq P(x) \leq 1 \) for all x
  3. \( \sum P(x) = 1 \)

Example 1

Flip a coin twice where \( x \) = number of tails

Identify the probability distribution for \( x \)

Let's determine the probability distribution:

Possible outcomes when flipping a coin twice: HH, HT, TH, TT

Where x = number of tails can be 0, 1, or 2:

  • \( x = 0 \) (HH): 1 outcome, \( P(x=0) = \frac{1}{4} = 0.25 \)
  • \( x = 1 \) (HT, TH): 2 outcomes, \( P(x=1) = \frac{2}{4} = 0.5 \)
  • \( x = 2 \) (TT): 1 outcome, \( P(x=2) = \frac{1}{4} = 0.25 \)

Probability Distribution Table:

\( x \)\( P(x) \)
00.25
10.5
20.25

Verification: \( \sum P(x) = 0.25 + 0.5 + 0.25 = 1 \), and all probabilities are between 0 and 1, so this is a valid probability distribution.

Example 2

Draw one card from a standard deck of 52 cards where \( x = 1 \) if the card is a face card (Jack, Queen, or King), \( x = 0 \) if it is not a face card

Identify the probability distribution for \( x \)

Let's determine the probability distribution:

A standard deck has 52 cards. Face cards are Jacks, Queens, and Kings (3 per suit, 4 suits):

  • Number of face cards = \( 3 \times 4 = 12 \)
  • Number of non-face cards = \( 52 - 12 = 40 \)

Where x = 1 for face cards and x = 0 for non-face cards:

  • \( x = 0 \) (non-face cards): 40 outcomes, \( P(x=0) = \frac{40}{52} = \frac{10}{13} \approx 0.7692 \)
  • \( x = 1 \) (face cards): 12 outcomes, \( P(x=1) = \frac{12}{52} = \frac{3}{13} \approx 0.2308 \)

Probability Distribution Table:

\( x \)\( P(x) \)
0\( \frac{10}{13} \approx 0.7692 \)
1\( \frac{3}{13} \approx 0.2308 \)

Verification: \( \sum P(x) = \frac{10}{13} + \frac{3}{13} = \frac{13}{13} = 1 \), and all probabilities are between 0 and 1, so this is a valid probability distribution.
