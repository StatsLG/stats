Density Curve

A density curve is a smooth curve that represents a population's distribution.

Statistical software can replace the separate bars of a histogram with a smooth curve (density curve) that represents the overall shape of a distribution.

The shape of a density curve is determined similarly to histograms.

Density curves help determine the proportion, percentage, or probability of the population falling within an interval. In the discrete case, probability is assigned to individual points, whereas in a continuous random variable, probability is assigned to intervals.

These curves are set up so that the area under the curve represents the proportion, percentage, or probability of observations.

For a curve to be a valid density curve representing a continuous probability distribution, it must satisfy the following properties:

  • The curve must be continuous (it represents a continuous random variable).
  • The curve must never fall below the x-axis (probability can never be less than 0).
  • The total area under the curve must be 1 (probability can never be greater than 1, and the total probability in any distribution must equal 1).
